# 🧩 Práctica 02: Creación de Recursos con POST

## 🎯 Objetivos

- Construir payloads JSON para crear recursos
- Configurar headers en peticiones POST
- Validar datos mínimos antes de enviar

## 📋 Descripción

En esta práctica implementarás operaciones `Create` usando `POST`, verificando respuesta HTTP y mostrando confirmación.

## 📝 Pasos

1. Construir payload
2. Enviar POST con headers
3. Validar antes de enviar
4. Crear función reusable

## 🚀 Instrucciones

1. Abre `starter/index.js`
2. Descomenta cada bloque por orden
3. Ejecuta `node starter/index.js`
4. Revisa el ID devuelto por la API
