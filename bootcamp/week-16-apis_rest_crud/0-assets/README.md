# 🎨 Assets - Semana 16

Recursos visuales para explicar APIs REST, operaciones CRUD y estados de carga.

## Archivos

- `01-rest-crud-map.svg`: Relación entre recursos REST y operaciones CRUD
- `02-http-methods-flow.svg`: Flujo típico de cliente API para métodos HTTP
- `03-loading-states.svg`: Estados de UI para operaciones asíncronas
- `04-auth-token-flow.svg`: Ciclo de autenticación con token y manejo de 401
- `05-error-handling-matrix.svg`: Matriz de clasificación de errores y acciones UX

## Uso pedagógico sugerido

1. Abrir `01-rest-crud-map.svg` en el bloque de fundamentos para mapear recurso + método.
2. Usar `02-http-methods-flow.svg` antes de prácticas CRUD para tomar decisiones de request.
3. Referenciar `03-loading-states.svg` y `05-error-handling-matrix.svg` durante diseño de estados.
4. Trabajar `04-auth-token-flow.svg` al explicar headers y Authorization.

## Criterio de cantidad de SVG

No hay límite fijo de SVG por semana. Se agregan tantos como aporten claridad conceptual, práctica y evaluable para los aprendices.

## Lineamientos visuales

- Tema dark
- Colores sólidos (sin gradientes)
- Alto contraste
- Tipografía sans-serif
