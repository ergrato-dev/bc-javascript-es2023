# 📖 Glosario Week 19 (A-Z)

## I

- **IndexedDB**: base de datos NoSQL integrada en el navegador, orientada a almacenamiento local asíncrono.

## J

- **JSON.parse**: convierte texto JSON en estructura JavaScript.
- **JSON.stringify**: convierte estructura JavaScript en texto JSON.

## L

- **localStorage**: almacenamiento persistente por origen, basado en pares clave-valor string.

## Q

- **QuotaExceededError**: error al intentar guardar datos superando el límite disponible.

## S

- **sessionStorage**: almacenamiento por pestaña/sesión con vida temporal.
- **Storage key**: identificador textual que referencia un dato en Web Storage.
