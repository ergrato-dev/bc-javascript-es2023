# 📖 Libros Electrónicos Gratuitos

Recursos bibliográficos gratuitos para profundizar en Arrays Avanzados de JavaScript.

---

## 📚 Libros Recomendados

### 1. Eloquent JavaScript (3ra Edición)

- **Autor**: Marijn Haverbeke
- **Capítulo relevante**: Chapter 4 - Data Structures: Objects and Arrays
- **Enlace**: [eloquentjavascript.net](https://eloquentjavascript.net/04_data.html)
- **Idioma**: Inglés (con traducción al español disponible)
- **Por qué leerlo**: Excelente explicación de estructuras de datos y métodos de arrays

### 2. JavaScript.info - Arrays

- **Autor**: Ilya Kantor
- **Sección relevante**: Data Types → Arrays / Array Methods
- **Enlace**: [javascript.info/array-methods](https://javascript.info/array-methods)
- **Idioma**: Inglés/Español
- **Por qué leerlo**: Guía completa con ejemplos interactivos de todos los métodos de arrays

### 3. You Don't Know JS Yet (2da Edición)

- **Autor**: Kyle Simpson
- **Libro relevante**: Get Started - Chapter 3
- **Enlace**: [github.com/getify/You-Dont-Know-JS](https://github.com/getify/You-Dont-Know-JS/tree/2nd-ed)
- **Idioma**: Inglés
- **Por qué leerlo**: Profundización en conceptos fundamentales de JavaScript

### 4. JavaScript Notes for Professionals

- **Autor**: GoalKicker.com
- **Secciones relevantes**: Arrays (Cap. 11)
- **Enlace**: [goalkicker.com/JavaScriptBook](https://goalkicker.com/JavaScriptBook/)
- **Idioma**: Inglés
- **Por qué leerlo**: Referencia rápida y completa de métodos de arrays

### 5. Exploring ES2023

- **Autor**: Dr. Axel Rauschmayer
- **Capítulo relevante**: 7.10 New Array methods
- **Enlace**: [exploringjs.com/es6](https://exploringjs.com/es6/ch_arrays.html)
- **Idioma**: Inglés
- **Por qué leerlo**: Detalle técnico de los nuevos métodos introducidos en ES2023

---

## 🎯 Orden de Lectura Sugerido

1. **JavaScript.info** - Para empezar (ejemplos prácticos)
2. **Eloquent JavaScript** - Fundamentos sólidos
3. **Exploring ES2023** - Detalles de ES2023
4. **You Don't Know JS** - Profundización avanzada
