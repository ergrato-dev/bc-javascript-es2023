# 📅 Semana 21: Fechas y Tiempos (Date API)

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, serás capaz de:

- ✅ Trabajar con el objeto `Date` para crear y leer fechas
- ✅ Formatear fechas y horas para interfaces de usuario
- ✅ Realizar operaciones con fechas (sumas, restas, diferencias)
- ✅ Comprender nociones clave de timezones
- ✅ Explorar introducción práctica a `Temporal API`
- ✅ Diseñar flujos confiables para agenda y calendario

---

## 📚 Requisitos Previos

Antes de iniciar esta semana, debes dominar:

- Formularios y validación (Semana 20)
- Web Storage APIs (Semana 19)
- Event handling avanzado (Semana 18)
