# 🗓️ Proyecto Semanal: Agenda Inteligente con Date API

> 🎯 ÚNICO ENTREGABLE: Este proyecto es el único entregable obligatorio para aprobar la semana.

## 🎯 Objetivo

Construir una mini aplicación de agenda usando `Date` para:

- Registrar eventos con fecha y hora
- Mostrar eventos ordenados por fecha
- Calcular tiempo restante para cada evento
- Formatear fechas para presentación al usuario

## 📋 Tu Dominio Asignado

Dominio: asignado por el instructor al inicio del trimestre.

Debes adaptar el concepto de "evento" a tu dominio.

## ✅ Requisitos Funcionales

1. Definir estructura de datos para ítems con fecha/hora.
2. Crear función de render para listar elementos.
3. Implementar ordenamiento por fecha.
4. Mostrar tiempo restante o estado (pendiente, próximo, vencido).
5. Formatear fechas con `toLocaleString`.

## 🧩 Estructura

- `index.html`: interfaz base
- `styles.css`: estilos
- `starter/script.js`: código con TODOs para completar
- `solution/`: referencia para instructores

## 🚀 Ejecución

1. Abre `index.html` en el navegador.
2. Implementa los TODOs en `starter/script.js`.
3. Verifica que la lista se renderice y se actualice correctamente.

## 🧠 Criterios de Evaluación

- Uso correcto de Date API
- Cálculo de diferencias de tiempo
- Ordenamiento consistente
- Código limpio y legible
- Adaptación clara al dominio asignado
