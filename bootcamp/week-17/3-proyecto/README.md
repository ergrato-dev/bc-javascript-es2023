# 🧱 Proyecto Semanal: Constructor de UI Dinámico

> **🎯 ÚNICO ENTREGABLE**: Este proyecto es el **único entregable obligatorio** para aprobar la semana.

## 🎯 Objetivo

Construir una interfaz dinámica que use manipulación moderna del DOM para:

- Seleccionar nodos con precisión
- Renderizar componentes reutilizables con `<template>`
- Gestionar estados visuales con `classList`
- Crear y remover nodos sin recargar la página

## 📋 Tu Dominio Asignado

**Dominio**: [El instructor te asignará tu dominio al inicio del trimestre]

Debes adaptar la UI y los nombres de entidad al dominio asignado.

## ⚠️ Nota sobre ejemplos

Se usan ejemplos neutrales (Museo, Acuario, Planetario) para no interferir con la política de dominios únicos.

## ✅ Requisitos Funcionales

1. Renderizar lista inicial de ítems
2. Agregar ítems nuevos desde formulario
3. Eliminar ítems existentes
4. Marcar ítems como destacados con `classList`
5. Mostrar estado de interfaz (idle/loading/success/error)
6. Usar `<template>` para tarjetas reutilizables
