# 🧱 Semana 17: DOM Manipulation Moderna

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, serás capaz de:

- ✅ Seleccionar nodos del DOM con `querySelector` y `querySelectorAll`
- ✅ Crear, insertar y remover elementos con APIs nativas
- ✅ Gestionar estilos de estado con `classList`
- ✅ Diferenciar cuándo usar `textContent` y cuándo `innerHTML`
- ✅ Renderizar bloques reutilizables con `<template>`
- ✅ Diseñar UIs dinámicas sin frameworks

---

## 📚 Requisitos Previos

Antes de iniciar esta semana, debes dominar:

- Fetch API y manejo de respuestas (Semana 15)
- APIs REST y CRUD (Semana 16)
- `async/await` y manejo de errores (Semanas 13-14)
- Arrays y objetos modernos de JavaScript (Semanas 2-5)
