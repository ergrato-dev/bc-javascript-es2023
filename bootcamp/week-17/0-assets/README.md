# 🎨 Assets - Semana 17

Recursos visuales pedagógicos para manipulación moderna del DOM.

## Archivos

- `01-dom-selection-map.svg`: mapa de selección con querySelector/querySelectorAll
- `02-node-lifecycle.svg`: ciclo de vida de nodo (crear, insertar, remover)
- `03-classlist-states.svg`: patrón de estados visuales con classList
- `04-textcontent-vs-innerhtml.svg`: comparación de uso y riesgo
- `05-template-workflow.svg`: flujo de render con template y cloneNode

## Criterio

La cantidad de SVG se define por valor pedagógico, no por un número fijo semanal.

## Lineamientos visuales

- Tema dark
- Colores sólidos sin degradados
- Tipografía sans-serif
- Prioridad en claridad didáctica
