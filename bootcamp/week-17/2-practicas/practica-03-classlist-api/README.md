# 🎛️ Práctica 03: classList API

## 🎯 Objetivos

- Agregar y remover clases de estado
- Validar clases activas con `contains`
- Aplicar patrón de estado exclusivo

## 📋 Descripción

Modelarás estados visuales (`idle`, `loading`, `success`, `error`) con una estructura simple y funciones reutilizables.

## 📝 Pasos

1. classList.add/remove
2. classList.toggle
3. classList.contains
4. setStatus centralizado

## 🚀 Instrucciones

1. Abre `starter/index.js`
2. Descomenta paso a paso
3. Ejecuta `node starter/index.js`
4. Verifica transición de estados
