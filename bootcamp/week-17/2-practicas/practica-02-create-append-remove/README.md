# 🧱 Práctica 02: createElement, append y remove

## 🎯 Objetivos

- Crear nodos dinámicamente
- Insertar nodos en lista simulada
- Eliminar nodos por criterio

## 📋 Descripción

Practicarás el ciclo básico de vida de nodos: crear, insertar y remover, manteniendo sincronía con estado en memoria.

## 📝 Pasos

1. Crear elemento
2. Insertar en colección
3. Remover por ID
4. Render completo desde estado

## 🚀 Instrucciones

1. Abre `starter/index.js`
2. Descomenta por pasos
3. Ejecuta `node starter/index.js`
4. Verifica el estado final
