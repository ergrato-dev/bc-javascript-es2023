# 🔎 Práctica 01: Selectores Modernos

## 🎯 Objetivos

- Usar `querySelector` y `querySelectorAll`
- Aplicar selectores por `data-*`
- Evitar consultas redundantes

## 📋 Descripción

Explorarás distintas estrategias de selección de nodos para construir una base sólida antes de manipular la interfaz.

## 📝 Pasos

1. Selección única
2. Selección múltiple
3. Selectores por data-attribute
4. Cache de nodos frecuentes

## 🚀 Instrucciones

1. Abre `starter/index.js`
2. Descomenta sección por sección
3. Ejecuta `node starter/index.js`
4. Verifica salida en consola
