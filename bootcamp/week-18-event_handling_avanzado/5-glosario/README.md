# 📖 Glosario Week 18 (A-Z)

## A

- **Action Map**: objeto que mapea acciones (`data-action`) a funciones.

## B

- **Bubbling**: fase donde el evento sube desde el target hacia ancestros.

## C

- **Capturing**: fase donde el evento baja desde ancestros hasta el target.
- **CustomEvent**: evento personalizado con datos en `detail`.

## D

- **Delegación de eventos**: patrón que usa un listener en elemento padre para manejar eventos de hijos.

## E

- **EventTarget**: interfaz base de objetos que pueden recibir y emitir eventos.

## F

- **Fase de evento**: etapa del flujo (capturing, target, bubbling).

## P

- **preventDefault()**: cancela comportamiento por defecto del navegador cuando es permitido.

## S

- **stopPropagation()**: detiene propagación del evento a otros nodos.
