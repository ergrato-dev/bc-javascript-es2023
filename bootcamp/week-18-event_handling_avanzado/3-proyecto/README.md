# 🏛️ Proyecto Semanal: Sistema de Notificaciones Interactivas

> 🎯 ÚNICO ENTREGABLE: este proyecto es el único entregable obligatorio para aprobar la semana.

## 🎯 Objetivo

Construir un sistema de notificaciones con JavaScript moderno aplicando:

- Delegación de eventos
- Control de flujo con `preventDefault` y `stopPropagation`
- Eventos personalizados con `CustomEvent`
- Buen manejo del ciclo de vida de listeners

## 📋 Tu dominio asignado

Dominio: asignado por el instructor (ej. Biblioteca, Farmacia, Gimnasio, etc.).

Adapta la semántica de las notificaciones a tu dominio.

## ✅ Requisitos funcionales

1. Renderizar una lista de notificaciones iniciales
2. Agregar nuevas notificaciones desde formulario
3. Cerrar una notificación individual
4. Marcar notificaciones como leídas/no leídas
5. Emitir al menos 2 `CustomEvent` de dominio
6. Mostrar métricas en tiempo real: total, leídas, pendientes

## 🧠 Requisitos técnicos

1. Usar `const`/`let`, arrow functions, template literals y destructuring
2. Implementar delegación de eventos en un contenedor
3. Evitar listeners duplicados
4. Mantener inmutabilidad básica en actualizaciones de estado
5. Incluir comentarios educativos en partes clave

## 🛠️ Estructura esperada

- `index.html`
- `styles.css`
- `starter/script.js`

## 🚀 Ejecución

1. Abre `index.html` con Live Server
2. Completa los TODOs en `starter/script.js`
3. Verifica funcionamiento manual de todos los flujos

3. Capturas de pantalla del sistema funcionando
