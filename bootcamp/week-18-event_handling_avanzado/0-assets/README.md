1. Presenta primero el diagrama sin código y pide hipótesis del grupo.
2. Usa las secciones internas del SVG (objetivo, lectura guiada y verificación).
3. Pasa a práctica con un caso mínimo ejecutable.
4. Cierra con una evidencia verificable (predicción de logs o mini implementación).
