# 📖 Glosario Week 20 (A-Z)

## C

- **Constraint Validation API**: API del navegador para validación programática de formularios.

## F

- **FormData**: interfaz para recolectar datos de formularios como pares clave-valor.

## H

- **HTML5 validation**: validación nativa basada en atributos del formulario.

## P

- **preventDefault()**: método para evitar comportamiento nativo del envío de formulario.

## R

- **reportValidity()**: método para mostrar mensajes de validación del navegador.

## S

- **setCustomValidity()**: método para definir mensajes personalizados de error en un campo.
- **Submit handling**: control lógico del proceso de envío de formulario.
