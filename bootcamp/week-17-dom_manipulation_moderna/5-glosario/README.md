# 📖 Glosario - Semana 17: DOM Manipulation Moderna

## A

### append
Método para insertar uno o más nodos al final de un contenedor.

## C

### classList
API para gestionar clases CSS de un elemento (`add`, `remove`, `toggle`, `contains`).

### cloneNode
Método que duplica un nodo; con `true` incluye descendientes.

## D

### DOM
Document Object Model: representación en árbol del documento HTML.

### data-attributes
Atributos `data-*` usados para almacenar metadatos en HTML.

## E

### Element
Nodo de tipo elemento en el DOM (`div`, `section`, `button`, etc.).

## F

### Fragment
Contenedor ligero para agrupar nodos antes de insertarlos.

## I

### innerHTML
Propiedad que permite leer/escribir HTML dentro de un nodo. Debe usarse con precaución.

## N

### Node
Unidad base del árbol DOM (elementos, texto, comentarios, etc.).

## P

### prepend
Método para insertar nodos al inicio de un contenedor.

## Q

### querySelector
Método que retorna el primer elemento que coincide con un selector CSS.

### querySelectorAll
Método que retorna un NodeList estático con todos los elementos que coinciden.

## R

### remove
Método para eliminar un nodo del DOM.

### render
Proceso de transformar estado/datos en nodos visibles.

## S

### Selector CSS
Patrón para localizar nodos (`.class`, `#id`, `[data-x]`, etc.).

## T

### template
Elemento HTML para definir estructura reutilizable no renderizada hasta ser clonada.

### textContent
Propiedad para leer/escribir texto plano en un nodo.

## U

### UI State
Estado de interfaz (idle, loading, success, error) reflejado visualmente.
