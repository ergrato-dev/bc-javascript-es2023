# 🧩 Práctica 04: Template y Render Dinámico

## 🎯 Objetivos

- Simular render con estructura tipo `<template>`
- Clonar y mapear datos a campos
- Renderizar listas sin duplicar markup manual

## 📋 Descripción

Practicarás la idea de componente reutilizable: una estructura base + datos dinámicos por item.

## 📝 Pasos

1. Definir template base
2. Clonar template
3. Mapear datos
4. Render de colección

## 🚀 Instrucciones

1. Abre `starter/index.js`
2. Descomenta cada paso
3. Ejecuta `node starter/index.js`
4. Analiza salida de render
