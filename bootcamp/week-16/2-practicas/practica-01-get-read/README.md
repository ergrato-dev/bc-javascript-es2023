# 📘 Práctica 01: Lectura de Recursos con GET

## 🎯 Objetivos

- Consultar colecciones y recursos individuales
- Aplicar filtros con `URLSearchParams`
- Renderizar resultados básicos en consola

## 📋 Descripción

Consumirás endpoints de lectura con `GET` sobre una API pública para dominar búsqueda, paginación y lectura por ID.

## 📝 Pasos

1. GET de colección
2. GET por ID
3. GET con query params
4. Función reusable para lectura

## 🚀 Instrucciones

1. Abre `starter/index.js`
2. Descomenta sección por sección
3. Ejecuta `node starter/index.js`
4. Verifica que cada paso imprima resultados esperados
