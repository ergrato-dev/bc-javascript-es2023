# 🧪 Práctica 04: DELETE, Auth Headers y Estados

## 🎯 Objetivos

- Eliminar recursos con `DELETE`
- Agregar header `Authorization`
- Modelar estados `loading`, `success`, `error`

## 📋 Descripción

Practicarás el flujo completo de eliminación con confirmación lógica, autenticación por token y feedback visual en consola.

## 📝 Pasos

1. DELETE básico
2. Headers de autenticación
3. Estado de UI
4. Reintento ante error

## 🚀 Instrucciones

1. Abre `starter/index.js`
2. Descomenta cada paso en orden
3. Ejecuta `node starter/index.js`
4. Observa cambios de estado en consola
