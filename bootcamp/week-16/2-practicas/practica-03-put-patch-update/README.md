# 🔄 Práctica 03: Actualización con PUT y PATCH

## 🎯 Objetivos

- Diferenciar actualización completa y parcial
- Implementar `PUT` y `PATCH`
- Reusar lógica de validación y envío

## 📋 Descripción

Actualizarás recursos existentes con dos estrategias: reemplazo completo (`PUT`) y actualización parcial (`PATCH`).

## 📝 Pasos

1. Actualización completa con PUT
2. Actualización parcial con PATCH
3. Verificar respuesta y estado
4. Función genérica update

## 🚀 Instrucciones

1. Abre `starter/index.js`
2. Descomenta sección por sección
3. Ejecuta `node starter/index.js`
4. Compara el payload enviado en PUT vs PATCH
