# 📚 Recursos - Semana 16

Recursos complementarios para reforzar APIs REST, métodos HTTP y diseño de flujos CRUD.

---

## 🎯 Cómo usar estos recursos

1. **Antes de prácticas**: repasa conceptos base de HTTP y REST en webgrafía.
2. **Durante prácticas**: usa herramientas de prueba para inspeccionar requests y responses.
3. **Antes del proyecto**: revisa ejemplos de arquitectura de cliente API y manejo de errores.
4. **Después del proyecto**: compara tu implementación con checklist de buenas prácticas.
