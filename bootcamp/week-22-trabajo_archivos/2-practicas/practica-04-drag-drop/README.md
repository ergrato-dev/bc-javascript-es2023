# 🧪 Práctica 04: Drag and Drop

## 🎯 Objetivos

- Implementar una zona de arrastre para archivos
- Manejar los eventos `dragenter`, `dragover`, `dragleave` y `drop`
- Dar feedback visual durante el arrastre
- Procesar los archivos recibidos

## 🚀 Instrucciones

1. Abre `starter/index.js`
2. Descomenta cada paso en orden
3. Ejecuta abriendo `index.html` en el navegador
4. Compara con `solution/index.js`
